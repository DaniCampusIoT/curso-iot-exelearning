Inicio

Proyecto ABP · REA · 1.º de Bachillerato

Diseña, programa y visualiza una estación meteorológica IoT.

Este recurso propone un proyecto tecnológico completo en el que el alumnado monta un sistema con ESP8266 y BMP280, obtiene datos del entorno, los transmite mediante MQTT en formato JSON y los representa en un dashboard funcional con Node-RED.

ESP8266 BMP280 MQTT JSON Node-RED 12–14 sesiones
Pregunta guía

¿Cómo podemos captar datos del entorno, enviarlos a través de la red y convertirlos en información útil mediante una interfaz visual en tiempo real?

Nivel 1.º Bach. Tecnología e Ingeniería con enfoque práctico y competencial.
Metodología ABP Aprendizaje basado en proyectos con producto funcional final.
Duración 12–14 Sesiones distribuidas entre práctica, teoría y evaluación.
Producto Dashboard Visualización de datos reales en Node-RED.

Flujo del sistema

①
Sensor BMP280 mide temperatura y presión.
→
②
ESP8266 Procesa datos y gestiona la conectividad WiFi.
→
③
MQTT / JSON Publica la información en red de forma estructurada.
→
④
Node-RED Recibe, interpreta y muestra datos en tiempo real.

Ruta de aprendizaje

1
Electrónica y montaje Identificación de componentes, conexiones y comprobación del circuito con seguridad.
2
Lectura local del sensor Uso de Arduino IDE, monitor serie e introducción al protocolo I2C.
3
IoT y comunicaciones Comprensión de la arquitectura IoT, la red WiFi, MQTT y el formato JSON.
4
Visualización de datos Construcción de flujos y paneles en Node-RED para interpretar datos empíricos.
5
Reflexión y mejora Análisis crítico sobre aplicaciones, privacidad, sostenibilidad y posibles mejoras del sistema.

Producto final esperado

Prototipo funcional

Una estación meteorológica IoT capaz de captar datos físicos del entorno, enviarlos por MQTT y representarlos en un dashboard visual para su análisis e interpretación.

Competencias que se activan

Pensamiento computacional, competencia digital, resolución de problemas tecnológicos y análisis de datos en contextos reales.
