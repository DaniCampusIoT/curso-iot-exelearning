Introducción al proyecto

Construimos una estación meteorológica IoT paso a paso.

En este proyecto el alumnado diseñará y pondrá en funcionamiento una estación meteorológica IoT capaz de captar datos del entorno, procesarlos con una placa ESP8266, leer un sensor BMP280, enviarlos mediante MQTT en formato JSON y representarlos en un dashboard creado con Node-RED.

A lo largo de las distintas fases se trabajará desde el montaje del hardware y la lectura local del sensor hasta la comunicación por red y la visualización final de los datos en tiempo real.

6 fases 12–14 sesiones ABP Dashboard final

Qué vas a construir

Un sistema real de adquisición, transmisión y visualización de datos que conecta sensor, microcontrolador, red, broker y panel de visualización.

Qué vas a aprender

Electrónica básica, lectura de sensores, comunicación WiFi, mensajería MQTT, estructura JSON y creación de flujos en Node-RED.

Cómo funciona el sistema

Sensor Mide datos del entorno.
ESP8266 Procesa y prepara la información.
MQTT Envía los datos por red.
JSON Estructura el mensaje.
Dashboard Muestra los datos en tiempo real.

Arquitectura general del sistema

Este sistema funciona como una cadena de trabajo conectada. Primero, el sensor BMP280 recoge datos del entorno, como la temperatura y la presión atmosférica. Después, la placa ESP8266 procesa esa información y la envía por WiFi mediante MQTT en formato JSON. Finalmente, Node-RED recibe los datos, los interpreta y los muestra de forma visual en un dashboard para que puedan analizarse fácilmente en tiempo real.
